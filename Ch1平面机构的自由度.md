# 1.1运动副及其分类

## 基本概念

1. 构件

   机构中最小的运动单元。

2. 运动副(***<u>是机械原理中最重要的概念之一</u>***)

   两个构件直接接触且具有确定相对运动的活动连接称为运动副。

   对运动副的理解要把握以下三点：

   + 运动副是一种活动连接
   + 运动副由两个构件组成
   + 组成运动副的两个构件之间有确定的相对运动(比如构件1和2之间一定是相对转动或移动)

3. 运动副元素(两构件参与接触构成运动副的部分)

   两构件相互接触处的几何形状(点、线、面)。

4. ***<u>机构是构件通过运动副连接形成的。</u>***

   也就是说，机构是由运动副逐一连接各个构件组成的。



## 分类

+ 按俩构件的接触形式分类

  面与面接触的运动副--低副

  点、线接触的运动副--高副

+ 按俩构件相对运动的形式分类

  平面运动副

  空间运动副

+ 按接触部分的几何形状分类

  球面副、螺旋副、圆柱副



根据组成运动副的两个构件的相对运动是否发生在同一平面内，分为平面运动副和空间运动副

## 运动链

1. 定义

   若干构件(两个或两个以上)通过运动副连接而成的可动系统

2. 若将运动链的一个构件相对固定(成为机架)，运动链则成为机构。

   PS：机构中各构件的相对运动是确定的，运动是不确定的！！

3. 机构中构件的分类

   + 机架（描述运动的参考系）
   + 原动件（运动规律已知的构件）
   + 从动件









# 1.2平面机构运动简图的绘制（1/5的概率考）

## 定义

机构运动简图：

在研究机构运动时，为了使问题简化，有必要撇开那些***<u>与运动无关的构件外形和运动副具体构造</u>***，仅用***<u>简单的线条和符号来表示构件和运动副</u>***，并按比例定出***<u>各运动副的位置</u>***。(老师提了四点注意点)

这种说明机构各构件间关系的简化图形，称为机构运动简图。它与原机构具有***<u>完全相同</u>***的运动特性。





用途：分析现有机构，构思设计新机械。

## 与运动有关的因素

1. 构件数目、运动副数目及类型
2. 运动副之间的相对位置



## 表达方式

1. 用简单线条表示构件
2. 规定符号代表运动副
3. 按比例定出运动副的相对位置
   + 转动副之间的中心距离
   + 移动副之间的导路中心线距离、夹角
   + 移动副导路中心线到转动副中心距离的距离
   + 点线副的外形

## 平面机构运动简图的绘制方法和步骤

1. 确定构件数目及原动件、输出构件。
2. 根据各构件间的相对运动确定运动副的种类和数目。
3. 选定比例尺，按规定符号绘制运动简图。
4. 标明机架、原动件和作图比例尺。
5. 验算自由度



顺口溜：

先两头，后中间，从头至尾走一遍，数数构件是多少，再看它们怎相连。

# 1.3平面机构自由度的计算(8分左右)

## 构件的自由度

1. 自由度--构件所具有独立运动的个数(***<u>确定构件位置所需独立坐标数</u>***)

2. 一个完全自由的平面运动构件，具有三个自由度。

   如下图所示

   ![img](https://wx1.sinaimg.cn/mw690/005LasY6gy1gcgm4fv3qvj306903ljrk.jpg)

   这个构件可以绕着原点进行转动，还可以沿着x、y轴进行移动，因此它的自由度F = 3。



## 平面运动副的约束条件

运动副的形成引入了约束，使构件失去运动自由度。

1. 转动副

   ![img](https://wx3.sinaimg.cn/mw690/005LasY6gy1gcgmb8hxmzj30az03ngm5.jpg)

   当一个构件上安装了一个转动副以后，这个构件不能沿着x、y轴移动，只能转动。

2. 移动副

   ![img](https://wx4.sinaimg.cn/mw690/005LasY6gy1gcgyhvhbpoj304o04rgls.jpg)

   当一个构件上安装了一个移动副之后，只能沿x轴移动，y轴的移动和转动都被限制了。

3. 齿轮副和凸轮副

   ![img](https://wx3.sinaimg.cn/mw690/005LasY6gy1gcgyi0tycqj306y04m74s.jpg)

   当一个构件上安装了齿轮副或者凸轮副之后，构件可以绕着接触点转动，也可以沿着接触点处切线方向移动。(不能沿法线移动)



## 平面机构的自由度

1. 平面机构的自由度计算公式

   F = 3 * 活动构件数 - 2 * 低副数 - 1 * 高副数

   即

   F = 3 * n - 2 * P(l) - 1 * P(h)

   其中，活动构件数就是除了机架以外的构件数。

## 机构(运动链)具有确定运动的条件

1. 有一个机架
2. F > 0
3. 原动件数 = 自由度数

***<u>通常，原动件为含低副构件且与机架相连，且只有一个自由度。</u>***
